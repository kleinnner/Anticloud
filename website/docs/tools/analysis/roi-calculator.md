# ROI Calculator

ROI Calculator models the financial returns of cloud infrastructure and security investments. It accounts for migration costs, operational savings, risk reduction, and productivity gains over configurable time horizons.

## Features

- Cost-Benefit Analysis: Input migration costs, operational savings, and productivity improvements
- Payback Period: Calculate break-even points with monthly and cumulative cash flow projections
- NPV Modeling: Discount future cash flows to present value with adjustable discount rates
- Sensitivity Analysis: Test how changes in key assumptions affect overall ROI projections
- Comparison Mode: Side-by-side evaluation of multiple investment scenarios or strategies

## Workflow

```mermaid

    B --> C[Financial Modeling]
    C --> D[Sensitivity Analysis]

    E --> F[Investment Recommendation]
```
