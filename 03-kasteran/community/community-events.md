# Kasteran* — Community Events
© Lois-Kleinner & 0-1.gg 2026

## Overview

Community events bring Kasteran* users and contributors together to learn, collaborate, and build. Events include hackathons, meetups, conferences, online workshops, and sprints. All events are designed to be inclusive, accessible, and valuable for participants of all skill levels.

## Hackathons

### Kasteran* Global Hackathon
- **Frequency**: Quarterly
- **Format**: Online, 48 hours
- **Participation**: Individual or teams of up to 4
- **Theme**: Varies (Green Code, Edge Innovation, Data Safety, etc.)
- **Judging**: Community voting + expert panel
- **Prizes**: Cash, cloud credits, hardware, recognition

### Past Hackathons
| Event | Theme | Participants | Projects |
|---|---|---|---|
| Hack 2026 Q1 | Green Code | 500+ | 120 |
| Hack 2025 Q4 | Edge Innovation | 450+ | 98 |
| Hack 2025 Q3 | Data Safety | 380+ | 85 |

### Hackathon Structure
```
Day 1:
  09:00 - Opening ceremony
  10:00 - Hacking begins
  16:00 - Mentor office hours
Day 2:
  10:00 - Mid-point check-in
  18:00 - Hacking ends
  19:00 - Presentations
  21:00 - Winners announced
```

## Meetups

### Local Meetups
Community-organized local meetups:
- **Format**: Monthly or quarterly
- **Content**: Talks, workshops, networking
- **Location**: Cities worldwide
- **Support**: Kasteran* provides speakers, swag, and funding

### Online Meetups
- **Format**: Monthly, virtual
- **Content**: Lightning talks, demos, Q&A
- **Recording**: Published on YouTube
- **Time**: Rotating time zones

### Starting a Meetup
1. Express interest in community channels
2. Find a venue (or use online)
3. Plan the agenda
4. Promote through community channels
5. Host the event
6. Share feedback with organizers

## Conferences

### Kasteran*Conf
Annual flagship conference:
- **Frequency**: Annually
- **Location**: Rotating (virtual option available)
- **Duration**: 3 days
- **Format**: Keynotes, talks, workshops, hackathon
- **Attendance**: 500-1000 participants

### Call for Proposals (CFP)
```
cfp:
  deadline: 3 months before conference
  tracks:
    - Language design
    - Tools and infrastructure
    - Community and ecosystem
    - Case studies
  selection: Double-blind review
  benefits: Travel, accommodation, speaker pass
```

### Scholarship Program
- Need-based scholarships for attendance
- Diversity scholarships for underrepresented groups
- Student scholarships with reduced rates

## Online Workshops

### Free Workshops
- **Format**: Live, interactive, 2-3 hours
- **Frequency**: Monthly
- **Topics**: Beginner to advanced
- **Recordings**: Available on YouTube

### Workshop Topics
| Level | Topic | Duration |
|---|---|---|
| Beginner | Introduction to Kasteran* | 2 hours |
| Beginner | Type System Fundamentals | 3 hours |
| Intermediate | Concurrency with Runes | 3 hours |
| Intermediate | WASM Deployment | 2 hours |
| Advanced | Compiler Internals | 4 hours |
| Advanced | Performance Optimization | 3 hours |

### Workshop Structure
```
15 min - Introduction and setup
45 min - Core content (Part 1)
10 min - Break
45 min - Core content (Part 2)
15 min - Q&A and exercises
```

## Sprints

### Contribution Sprints
- **Format**: 1-2 day focused coding sessions
- **Frequency**: Monthly online, quarterly in-person
- **Focus**: Specific features, bugs, or documentation
- **Mentoring**: Experienced contributors mentor newcomers

### Sprint Types
- **Bug squashing**: Fixing open bugs
- **Documentation**: Improving docs and translations
- **Feature implementation**: Building new features
- **Infrastructure**: CI/CD, tooling improvements

## Code of Conduct

All events follow the Kasteran* Code of Conduct:
- Inclusive and welcoming environment
- Harassment-free experience
- Reporting process available
- Enforcement by trained moderators

## Accessibility

Events are designed to be accessible:
- Captioning for talks and workshops
- Sign language interpretation (upon request)
- Virtual participation option
- Scholarship program for financial barriers
- Diverse speakers and topics

## Conclusion

Kasteran* community events provide opportunities for learning, networking, and contribution. Whether you are a beginner or expert, there are events designed for you. Join us at the next event!
