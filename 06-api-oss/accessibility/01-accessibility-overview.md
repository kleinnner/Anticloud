# Accessibility Overview

## Overview

API-OSS is committed to making AI infrastructure accessible to everyone.

## Commitment

```
We follow WCAG 2.1 AA standards for our web interface.
Our CLI and API are designed for screen reader compatibility.
Documentation is provided in accessible formats.
```

## Standards

| Standard | Level | Status |
|---|---|---|
| WCAG 2.1 | AA | Compliant |
| Section 508 | US | Compliant |
| EN 301 549 | EU | Compliant |
