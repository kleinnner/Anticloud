# Keyboard Shortcuts

## Overview

Keyboard shortcuts for the API-OSS dashboard.

## Global Shortcuts

| Shortcut | Action |
|---|---|
| `?` | Show keyboard shortcuts help |
| `/` | Focus search bar |
| `g h` | Go to home/dashboard |
| `g k` | Go to API keys |
| `g r` | Go to routes |
| `g m` | Go to monitoring |
| `g s` | Go to settings |

## Navigation

| Shortcut | Action |
|---|---|
| `Tab` | Next element |
| `Shift+Tab` | Previous element |
| `Enter` | Activate/select |
| `Escape` | Close modal/cancel |
| `↑ ↓` | Navigate lists |
| `Space` | Toggle select |
| `Ctrl+f` | Search |

## Actions

| Shortcut | Action |
|---|---|
| `n` | Create new |
| `e` | Edit selected |
| `Delete` | Delete selected |
| `r` | Refresh |
| `Ctrl+s` | Save |
| `Ctrl+z` | Undo |
| `Ctrl+Shift+z` | Redo |
