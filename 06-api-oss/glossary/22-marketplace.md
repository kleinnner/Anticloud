# Glossary 22: Marketplace Glossary

## Terms

### Marketplace
- Central platform for discovering and installing plugins
- Accessible via web UI and CLI

### Listing
- A plugin or model published on the marketplace
- Includes: metadata, screenshots, pricing, reviews

### Publisher
- Individual or organization publishing to marketplace
- Must register and verify identity

### Revenue Share
- Percentage split of sale revenue
- Standard: 70% author, 30% platform

### Payout
- Transfer of earned revenue to publisher
- Monthly via: Stripe, PayPal, bank transfer

### Pricing Tier
- Different price points for a listing
- Options: free, one-time, subscription, usage-based

### Free Plugin
- Plugin available at no cost
- Often: trial version, limited features, or fully open-source

### Paid Plugin
- Plugin with a purchase price
- May be: one-time payment, subscription, or usage-based

### Plugin Rating
- User feedback score (1–5 stars)
- Influences search ranking and discovery

### Plugin Review
- Written feedback from users
- Moderation policy prevents spam and abuse

### Verified Publisher
- Publisher with verified identity
- Badge shown on listings; higher trust

### Featured Listing
- Prominently displayed plugin on marketplace
- Selected by editorial team or algorithmic

### Category
- Classification for marketplace listings
- Types: tool, connector, bridge, dashboard, data source, model

### Search (Marketplace)
- Finding plugins by keyword, category, price
- API-OSS marketplace search with filters

### Install Count
- Number of times a plugin has been installed
- Public metric shown on listing

### Compatibility
- API-OSS version requirements for a plugin
- Listed in plugin manifest
