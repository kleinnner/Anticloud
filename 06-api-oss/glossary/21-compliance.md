# Glossary 21: Compliance Glossary

## Terms

### GDPR (General Data Protection Regulation)
- EU regulation protecting personal data
- API-OSS local-first architecture supports GDPR compliance

### HIPAA (Health Insurance Portability and Accountability Act)
- US healthcare data privacy regulation
- API-OSS provides HIPAA compliance package

### SOX (Sarbanes-Oxley Act)
- US financial reporting and audit standard
- API-OSS audit ledger supports SOX compliance

### SOC 2 (Service Organization Control 2)
- Auditing standard for service providers
- API-OSS supports SOC 2 Type II requirements

### PCI-DSS (Payment Card Industry Data Security Standard)
- Security standard for payment card data
- API-OSS encryption and audit support PCI-DSS compliance

### LGPD (Lei Geral de Proteção de Dados)
- Brazilian data protection law
- Similar to GDPR in scope and requirements

### CCPA (California Consumer Privacy Act)
- California data privacy law
- API-OSS supports CCPA data deletion and access rights

### ISO 27001
- Information security management standard
- API-OSS can be deployed in ISO 27001 certified environments

### Data Residency
- Requirement that data stays within geographic boundaries
- API-OSS supports: EU-only, US-only, country-specific deployment

### DPA (Data Processing Agreement)
- Contract defining data processing responsibilities
- API-OSS provides DPA template for enterprise customers

### DPIA (Data Protection Impact Assessment)
- Risk assessment for data processing activities
- API-OSS provides documentation to support DPIA

### Data Retention Policy
- Rules for how long data is kept
- API-OSS supports configurable retention periods

### Data Deletion / Right to be Forgotten
- GDPR article 17 right to erasure
- API-OSS supports complete user data deletion

### Breach Notification
- Mandatory reporting of data breaches
- API-OSS audit ledger provides breach detection capability

### Audit Trail
- Immutable record of all operations
- API-OSS SHA-256 linked chain ensures tamper evidence

### Certification / Accreditation
- Third-party verification of compliance
- API-OSS supports compliance certification processes
