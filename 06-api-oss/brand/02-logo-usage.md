# Logo Usage

## Overview

Guidelines for using the API-OSS logo.

## Logo Variations

```yaml
primary:
  file: apioss-logo.svg
  usage: Primary brand logo
  clear_space: 50% of logo height

icon:
  file: apioss-icon.svg
  usage: App icons, avatars
  minimum_size: 24px

horizontal:
  file: apioss-logo-horizontal.svg
  usage: Headers, banners

monochrome:
  file: apioss-logo-white.svg
  usage: Dark backgrounds
```

## Proper Usage

```yaml
do:
  - Use provided files only
  - Maintain aspect ratio
  - Keep clear space
  - Use appropriate color variant

donot:
  - Stretch or distort
  - Change colors
  - Add effects (shadows, gradients)
  - Rotate
  - Overlay on busy backgrounds
```
