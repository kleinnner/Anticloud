# Color Palette

## Overview

Official API-OSS color palette.

## Primary Colors

```yaml
indigo:
  hex: "#6366F1"
  rgb: "99, 102, 241"
  hsl: "239, 84%, 67%"
  usage:
    - Primary brand color
    - Buttons and links
    - Interactive elements

indigo_dark:
  hex: "#4F46E5"
  rgb: "79, 70, 229"
  usage:
    - Hover states
    - Active elements
```

## Secondary Colors

```yaml
emerald:
  hex: "#10B981"
  rgb: "16, 185, 129"
  usage:
    - Success states
    - Positive metrics
    - Confirmation

amber:
  hex: "#F59E0B"
  rgb: "245, 158, 11"
  usage:
    - Warnings
    - Medium alerts

red:
  hex: "#EF4444"
  rgb: "239, 68, 68"
  usage:
    - Errors
    - Critical states
```

## Neutral Colors

```yaml
slate_900:
  hex: "#0F172A"
  usage: Background (dark mode)

slate_800:
  hex: "#1E293B"
  usage: Cards, surfaces (dark)

slate_50:
  hex: "#F8FAFC"
  usage: Background (light mode)

slate_100:
  hex: "#F1F5F9"
  usage: Cards, surfaces (light)
```

## Accessibility

```yaml
contrast_ratios:
  - indigo on white: 6.5:1 ✅
  - white on indigo: 6.5:1 ✅
  - slate_900 on slate_50: 15:1 ✅
  - emerald on white: 4.2:1 ✅
```
