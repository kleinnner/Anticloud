# Brand Templates

## Overview

Templates for branded API-OSS materials.

## Slide Deck Template

```yaml
format: 16:9 widescreen
colors:
  - Background: slate_50 (#F8FAFC)
  - Headings: indigo (#6366F1)
  - Body: slate_900 (#0F172A)
fonts:
  - Headings: Inter Bold
  - Body: Inter Regular
logo: apioss-logo-horizontal.svg (top right)
```

## Document Template

```yaml
format: Letter / A4
margins: 1 inch
colors:
  - Primary: indigo (#6366F1)
  - Body: slate_900 (#0F172A)
fonts:
  - Title: Inter Bold, 24pt
  - Headings: Inter Bold, 16pt
  - Body: Inter Regular, 11pt
  - Code: JetBrains Mono, 9pt
```

## Email Template

```yaml
max_width: 600px
colors:
  - Header: indigo (#6366F1)
  - Body: slate_900 (#0F172A)
  - Links: indigo (#6366F1)
fonts: Arial, Helvetica, sans-serif
footer:
  - Company name and address
  - Unsubscribe link
  - Social media links
```
