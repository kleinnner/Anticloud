# GENERAL — FREQUENTLY ASKED QUESTIONS

## What is API-SOS?

API-SOS (Agent-Predictive Intelligence Sovereign Operating System) is a local-first AI decision engine for regulated institutions. It runs entirely on hardware under your control with no cloud dependency.

## How is it different from ChatGPT / Claude / Gemini?

Those are cloud products that require internet, send your data to US servers, and can be cut off by foreign government action. API-SOS has zero cloud dependency — no data leaves your hardware, no API keys, no subscription per token.

## Is it open source?

Yes. The core source code is available on GitHub under Apache 2.0.

## Does it need internet?

No. API-SOS is designed to operate completely offline. Internet is only needed for the initial download of software and model files.

## What does "sovereign AI" mean?

Sovereign AI means the system runs on hardware you own, in a jurisdiction you control, with no dependency on foreign cloud infrastructure, foreign APIs, or foreign data centres. No foreign government can turn it off.

## What is the pricing?

Personal: $39/month or $399/year. Business: $49/seat/year. Enterprise: Custom ($50K-500K/year). Institutional: $1M+. No per-token charges because there are no API calls.

## Can I try it before buying?

The software is fully functional on your own hardware. You provide your own model files (GGUF format) and the system runs with zero ongoing API costs.

## What hardware do I need?

Minimum: 8 GB RAM, 4-core CPU, 10 GB storage. Recommended: 16 GB RAM, 8-core CPU, NVIDIA CUDA GPU with 8 GB+ VRAM, 50 GB SSD.

## What if the company goes out of business?

Your instance continues working indefinitely. It has zero dependency on our servers. The open-source code means anyone can fork and maintain it.
