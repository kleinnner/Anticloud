# Community Playbook 4: Recognition & Rewards

## Overview

How to recognize and reward community contributions effectively.

## Recognition Tiers

### Tier 1: First-time Contributors

```yaml
Action: First PR merged
Recognition:
  - Thank-you comment on PR
  - Added to ALL-CONTRIBUTORS.md
  - Discord role: "Contributor"
  - Sticker pack (mailed)
Cost: ~$5 per contributor (stickers)
```

### Tier 2: Regular Contributors

```yaml
Action: 5+ PRs merged (or significant docs/issues)
Recognition:
  - Name in README contributors section
  - Discord role: "Regular Contributor"
  - Contributor badge on website
  - Exclusive Discord channel access (#contributors)
  - T-shirt + hoodie (mailed)
Cost: ~$50 per contributor
```

### Tier 3: Core Contributors

```yaml
Action: 20+ PRs or maintainer role
Recognition:
  - GitHub commit bit
  - Profile on api-oss.local/community
  - Annual contributor summit invite (all expenses paid)
  - Voting rights on RFCs
  - Monthly stipend (if budget allows)
  - Priority support for their projects
Cost: ~$1,000–$5,000 per year
```

### Tier 4: Hall of Fame

```yaml
Action: Exceptional, sustained contribution
Recognition:
  - Named after them? (e.g., "The [name] Library")
  - Lifetime maintainer status
  - Speaking slot at conferences (sponsored)
  - Limited edition swag
  - Stock options (if incorporated)
Cost: Variable
```

## Non-Monetary Recognition

### Public Shoutouts

```yaml
Weekly: 
  - Discord #shoutouts channel
  - Twitter/LinkedIn (tag + thank you)
  - GitHub issue/PR comment

Monthly:
  - "Contributor of the Month" blog post
  - Newsletter feature

Quarterly:
  - "Hall of Fame" post
  - Live recognition at community standup
```

### Gamification

| Element | Implementation |
|---------|---------------|
| Points | 10 per PR merged, 5 per issue triaged, 2 per helpful comment |
| Badges | First PR, Bug Hunter, Docs Hero, Plugin Creator, Mentor |
| Leaderboard | Monthly top 10 contributors (public) |
| Levels | New → Contributor → Regular → Core → Maintainer |
| Quests | "Fix 5 documentation issues this month" with bonus points |

## Budget Guide

| Item | Monthly Cost | Annual Cost |
|------|-------------|-------------|
| Stickers + mail | $100 | $1,200 |
| T-shirts/hoodies | $200 | $2,400 |
| Supporter swag packs | $500 | $6,000 |
| Hackathon prizes | $500 | $2,000 (4×) |
| Summit travel (5 people) | — | $15,000 |
| Stipends (3 core × $500/mo) | $1,500 | $18,000 |
| **Total** | **$2,800** | **$44,600** |

## Simple Rule

**Always thank contributors publicly, ideally within 24 hours of their contribution.**
