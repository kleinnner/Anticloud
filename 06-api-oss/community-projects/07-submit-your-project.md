# Submit Your Project

## Overview

Submit your API-OSS community project.

## Submission Guidelines

```yaml
requirements:
  - Open source license (MIT, Apache 2.0)
  - Working integration with API-OSS
  - README with installation and usage
  - At least one user other than author
```

## How to Submit

```markdown
1. Fork the community projects repo
   https://github.com/example/community-projects

2. Add your project to projects.yml

3. Create a PR

4. Community review (7 days)

5. Approved → Listed in directory
```

## PR Template

```yaml
name: my-project
description: Description of your project
author: GitHub handle
url: https://github.com/author/my-project
category: integration
license: MIT
tags:
  - discord
  - bot
```

## Benefits

```yaml
- Listed in community directory
- API-OSS social media promotion
- Community contributor badge
- Invitation to contributor calls
- Early access to new features
```
