# Community Council

## Overview

The Community Council represents community interests in API-OSS governance.

## Purpose

```
- Advocate for community needs
- Organize community events
- Manage community programs
- Mediate disputes
- Represent user perspectives
```

## Composition

```
- 5 members
- Elected by contributors
- 6-month terms
- Max 2 consecutive terms
```

## Programs

| Program | Description |
|---|---|
| Mentorship | Guide new contributors |
| Hackathons | Community coding events |
| Local meetups | Regional gatherings |
| Ambassador program | Evangelists and advocates |
| Translation | Localize documentation |

## Meetings

```
Frequency: Monthly
Format: Public
Agenda: Community-sourced
```

## Communication

```
Discord: #community-council
GitHub: community-council team
Email: council@api-oss.local
```

## Budget

```
Annual community budget: $50,000
Allocation:
  - Events: 40%
  - Recognition: 25%
  - Infrastructure: 20%
  - Miscellaneous: 15%
```
