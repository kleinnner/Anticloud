# Code of Conduct Enforcement

## Overview

Enforcement procedures for the API-OSS Code of Conduct.

## Code of Conduct

We are committed to providing a welcoming, inclusive, and harassment-free experience for everyone.

## Enforcement Responsibilities

```
Community managers: First response (24h)
TSC: Escalation and appeals
Project lead: Final authority
```

## Reporting

```markdown
Report via:
- Email: conduct@api-oss.local
- GitHub: conduct@api-oss.local
- Private message to TSC members
```

## Enforcement Process

### Level 1: Warning

```
Behavior: Minor infraction (e.g., inappropriate language)
Action: Private warning
Duration: 30 days
```

### Level 2: Temporary Ban

```
Behavior: Repeated infractions, harassment
Action: 30-day ban from all community spaces
Duration: 30 days
```

### Level 3: Permanent Ban

```
Behavior: Severe violation (e.g., threats, doxxing)
Action: Permanent ban
Duration: Permanent
```

## Appeals

```
- File appeal within 7 days
- TSC reviews within 14 days
- Decision is final
```

## Transparency

```
- Aggregated reports shared quarterly
- Individual reports kept confidential
- Enforcement actions anonymized
```
