# Speaking Proposal Templates

## Overview

Templates for conference speaking proposals.

## Template

```markdown
## Title
[Engaging title for the talk]

## Abstract
2-3 paragraphs describing the talk content.

## Takeaways
- Takeaway 1
- Takeaway 2
- Takeaway 3

## Audience Level
Beginner / Intermediate / Advanced

## Duration
20 / 30 / 45 minutes

## Speaker Bio
2-3 sentences about the speaker.

## Previous Speaking
- Event 1 (Year)
- Event 2 (Year)
```

## Sample Proposals

### Open Source Summit

```markdown
Title: Building Sovereign AI Infrastructure with Open Source

Abstract: Organizations want to use AI but can't trust
commercial providers with their data. This talk presents
API-OSS, an open-source AI gateway that gives organizations
complete control over their AI infrastructure.

Takeaways:
- How to deploy AI infrastructure on your own cloud
- Multi-provider strategies for reliability
- Security patterns for compliance
```

### KubeCon

```markdown
Title: Running AI Gateways on Kubernetes at Scale

Abstract: How to deploy and operate AI API gateways
on Kubernetes, including auto-scaling, federation,
and multi-cluster deployments.

Takeaways:
- Helm chart patterns for AI gateways
- Auto-scaling with KEDA
- Multi-region federation
```
