# Press Kit

## Overview

Media resources for API-OSS.

## Company Description

```text
API-OSS is the leading open-source AI gateway platform. 
Organizations use API-OSS to deploy and manage AI infrastructure 
with complete data sovereignty, supporting any AI model provider 
through a unified API.

Founded in 2024, API-OSS is used by enterprises, governments, 
and defense organizations across 30+ countries.
```

## Boilerplate

```text
API-OSS (api-oss.local) is an open-source API gateway and AI platform 
that gives organizations complete control over their AI infrastructure. 
With OpenAI-compatible APIs, multi-provider routing, enterprise-grade 
security, and P2P federation, API-OSS enables any organization to 
deploy sovereign AI infrastructure without sacrificing capability.
```
