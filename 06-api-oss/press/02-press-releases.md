# Press Releases

## Release History

### v3.0.0 — April 15, 2025

**API-OSS Launches v3.0 with P2P Federation and WASM Plugin Runtime**

API-OSS, the leading open-source AI gateway, today announced the release of version 3.0.0, introducing peer-to-peer federation mesh networking, WASM plugin runtime, and AI-native features including model routing with A/B testing and semantic caching.

"Version 3.0 represents a fundamental shift in how organizations deploy AI infrastructure," said the API-OSS team. "With P2P federation, organizations can now run AI gateways across multiple regions and clouds with automatic failover."

### v2.0.0 — October 1, 2024

**API-OSS v2.0 Introduces Multi-Tenant Architecture and RBAC**

API-OSS announced version 2.0.0 with multi-tenant architecture, role-based access control, a redesigned admin API, and enhanced security features including JWT authentication and OAuth2/OIDC integration.

### v1.0.0 — June 15, 2024

**API-OSS Launches Open-Source AI Gateway**

API-OSS launched its initial release, providing an open-source alternative to commercial AI API gateways with full OpenAI API compatibility, rate limiting, authentication, and plugin system.
