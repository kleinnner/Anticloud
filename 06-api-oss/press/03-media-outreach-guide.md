# Media Outreach Guide

## Overview

Guidelines for media and PR outreach for API-OSS.

## Target Publications

### Tier 1 (Tech Industry)

```
- TechCrunch
- The Verge
- Wired
- Ars Technica
- ZDNet
```

### Tier 2 (Developer)

```
- Hacker News
- Dev.to
- InfoQ
- The New Stack
- Stack Overflow Blog
```

### Tier 3 (Enterprise/Industry)

```
- CIO.com
- Enterprise AI
- AI Business
- VentureBeat
- Forbes Tech
```

## Pitch Template

```markdown
Subject: API-OSS: Open-source AI gateway solves data sovereignty

Hi [Journalist],

API-OSS is an open-source AI gateway that lets organizations
deploy OpenAI-compatible APIs on their own infrastructure.

Key angles:
1. Data sovereignty without compromising on AI capability
2. Open-source alternative to proprietary AI APIs
3. Multi-provider routing with P2P federation

Would you be interested in a demo or interview?

Best,
[Name]
```

## Key Messages

```
1. Data sovereignty is the #1 concern for enterprise AI adoption
2. API-OSS provides commercial-grade AI without vendor lock-in
3. Open source enables customization and auditability
4. P2P federation enables global scale
```
