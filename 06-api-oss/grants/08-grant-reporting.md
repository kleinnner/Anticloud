# Grant Reporting

## Overview

Reporting requirements for grant-funded projects.

## Report Types

| Type | Frequency | Audience |
|---|---|---|
| Progress | Quarterly | Funder |
| Financial | Quarterly | Funder |
| Technical | Milestone-based | Funder |
| Final | Project end | Funder |

## Progress Report Template

```markdown
# Progress Report: [Grant Name]
Period: [Start] – [End]

## Executive Summary
Brief overview of progress.

## Milestones Achieved
- [ ] M1: [...]
- [ ] M2: [...]

## KPIs
| KPI | Target | Actual |
|---|---|---|
| Users | 100 | 85 |

## Budget
| Category | Allocated | Spent |
|---|---|---|
| Personnel | €100K | €45K |

## Next Steps
Plans for next reporting period.

## Issues & Risks
Any blockers or concerns.
```
