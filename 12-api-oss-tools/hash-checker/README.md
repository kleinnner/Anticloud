# Hash Checker

Cryptographic hash generation, verification, and checksum tool.

        C[Comparator]
    end
    subgraph Output["Output"]
        RH[Result Hash]
        VR[Verification Result]
        M[Match Status]
    end

    T --> G
    G --> C --> M
    H --> V --> VR
