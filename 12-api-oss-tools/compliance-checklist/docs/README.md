# Compliance Checklist

## Introduction

Controls review tracker with PDF export.

This tool demonstrates compliance requirement tracking and audit preparation -- a core capability of the API-OSS sovereign AI platform.

## Prerequisites

- Node.js 18+
- npm or yarn
- A modern web browser

## Quick Start

```bash
git clone https://github.com/kleinnner/api-oss-web.git
cd compliance-checklist
npm install
npm run dev
```
