# Regex Playground

Test and debug regular expressions in real-time. Everything runs in your browser — no data is sent to any server.

**Features:** Real-time pattern matching, capture group inspection, replacement preview, flag toggles (g, i, m, s, u), match position highlighting.

**Built with:** Next.js 15, React 19, Tailwind CSS, Framer Motion

**Deployment:** Deploy as a static site on Vercel, or run locally with `npm run dev`.
