# Frequently Asked Questions

## General

**Q: What does this tool demonstrate?**

A: Data localization scoring and sovereignty assessment.

**Q: Does this require an internet connection?**

A: The tool loads dependencies from CDNs on first visit. After caching, it works offline.

**Q: Does this tool work without API-OSS?**

A: Yes. All browser tools are standalone demonstrations.

## Technical

**Q: Are there privacy concerns?**

A: No. All computation happens in your browser. No data is sent to any server.

**Q: What browsers are supported?**

A: Chrome 110+, Edge 110+, Safari 16.4+, Firefox 120+.

**Q: How to report a bug?**

A: Open an issue at https://github.com/kleinnner/api-oss-web/issues

## Troubleshooting

**Q: The tool does not load.**

A: Clear your browser cache and reload. Check your internet connection.

**Q: I see a blank screen.**

A: Your browser may not support the required APIs. Try Chrome or Edge.

**Q: The tool is slow.**

A: Some tools load large WASM binaries on first visit (15MB for Python Playground). This is cached after the first load.

## Licensing
