# Compliance Generator

## Introduction

Generate compliance PDFs for procurement.

This tool demonstrates automated compliance document generation for procurement and audit workflows.

## Prerequisites

- Node.js 18+
- npm or yarn
- A modern web browser

## Quick Start

```bash
git clone https://github.com/kleinnner/anticloud.git
cd compliance-generator
npm install
npm run dev
```
