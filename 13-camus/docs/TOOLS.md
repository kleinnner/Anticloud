# Tools

## /graphify bar A:10 B:25 C:40

ASCII bar chart with plotext. Supports: bar, line, pie, scatter.

## /search <query>

Web search via DuckDuckGo. Returns up to 3 results. No API key needed.

## /status

System info: model, context, threads, vision, history count.

## /settings

Current configuration values from camus.json.

## /save <name> / /load <name>

Session persistence as JSON files in models/ directory.

## /clear

Clear conversation history.

## /help

Display all commands.
